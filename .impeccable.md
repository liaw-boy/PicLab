# Design Context — 照片白邊工具

## Users
攝影愛好者與創作者，目的是為作品快速加上精緻的相框並分享到 Instagram。他們同時是 Lightroom 使用者、也是 Copicseal 這類工具的受眾。使用情境：桌面，批次處理多張照片。

## Brand Personality
**專業 · 俐落 · 現代**
工具感強，不花俏。讓使用者覺得「這個工具懂我」，操作後的成品讓人有成就感。

## Aesthetic Direction
- 深色 UI（預設）+ 淺色 UI（可切換）— 跟隨 Copicseal 的深色調性但支援 Light mode
- 輸出圖片：白底、高端、適合各種使用者（攝影作品集 / 社群分享 / 自訂）
- 參考：Copicseal（深色多模板）、Lightroom（功能面板）、Apple Photos（視覺細膩度）
- 反例：避免過度卡通化、避免純黑設計（要有層次感）

## Design Principles
1. **輸出優先** — UI 是工具，輸出圖才是作品。UI 不搶眼，讓預覽圖發光
2. **即時反饋** — 每個動作都有視覺確認（動畫、spinner、狀態列）
3. **層次清晰** — 設定面板用卡片 + 標題建立視覺群組，不讓使用者迷路
4. **深淺一致** — 深色/淺色切換時，所有元件保持同等精緻度
5. **字型為王** — 輸出圖的 EXIF 文字需用支援 CJK 的高品質字型

## Output Image Standards
- 白底（RGB 255,255,255）
- 字型：Microsoft JhengHei / Noto Sans TC 優先（支援中文標籤）
- 分隔符：中點 `·`，不用 pipe `|`
- 品牌文字：各品牌用對應顏色，透明底，融入白邊
